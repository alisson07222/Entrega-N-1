
# Conservación y Naturaleza

## Descripción
Sitio web educativo dedicado a la concientización sobre los animales salvajes, su importancia en el ecosistema y las razones por las cuales no deben ser domesticados.

## Tecnologías usadas
- HTML5
- CSS3
- Git
- GitHub

## Instalación / Visualización
1. Clonar el repositorio:
   git clone https://github.com/alisson07222/Entrega-N-1.git
2. Abrir el archivo index.html en el navegador.
